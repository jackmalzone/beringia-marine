# How to Create Internal Links in Sanity Studio

## Quick Steps

1. **Select the text** you want to link (e.g., "Vital Ice" or "red light therapy session")

2. **Click the link icon** in the toolbar (🔗) or press `Cmd+K` (Mac) / `Ctrl+K` (Windows)

3. **Enter the internal path** in the URL field:
   - Use relative paths starting with `/`
   - Examples:
     - Homepage: `/`
     - Services: `/services/red-light-therapy`
     - Insights: `/insights`
     - Experience: `/experience`

4. **Leave "Open in new tab" unchecked** (for internal links)

5. **Click Save/Apply**

## Tips

- **Internal links** use relative paths (start with `/`)
- **External links** use full URLs (start with `http://` or `https://`)
- Internal links typically should NOT open in a new tab (better UX)
- External links can optionally open in a new tab
